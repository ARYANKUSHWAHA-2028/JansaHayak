# JansaHayak

A Django-based Government Scheme Finder that helps users discover government schemes based on eligibility criteria.

## Features

- User-friendly questionnaire
- Eligibility-based scheme recommendation
- Government scheme listing
- Scheme detail pages
- Responsive UI using Bootstrap
- Django backend integration
- Database-driven scheme management

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
